# Physics-Web
Project ini merupakan salah satu website yang diperuntukan untuk lomba soedirmantechnophoria yang bertemakan website berbentuk forum untuk menyelesaikan masalah yang ada di lingkungan kita. MyPhysics menawarkan solusi sebagai forum untuk mahasiswa fisika Indonesia yang kesulitan untuk memahami konsep-konsep fisika. Forum ini menyediakan fitur-fitur yang mendukung hal tersebut seperti fitur formula matematika untuk memudahkan pengguna untuk menulis rumus fisika yang kompleks.
# Fitur
1. Autenkasi seperti Sign in, Log in, dan Log out.
2. Menambahkan postingan dengan input thumbnail, judul, caption (disertai dengan fitur penambahan formula), dan refrensi (fitur menambah sitasi otomatis dengan mencantumkan link DOI)
3. Komentar dilengkapi dengan fitur penambahan formula matematika untuk forum yang komunikatif
4. Pengklasifikasian data postingan otomatis dengan algoritma ML menjadi tiga kategori : pertanyaa,konsep, dan penelitian fisika.
